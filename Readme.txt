O objetivo deste estudo é coletar dados financeiros de fontes confiáveis para análise de indicadores
    e tomada de decisão de compra/venda de ativos, adquirir/manter/vender uma posição ou apenas estudo
    do mercado.



Os arquivos com prefixo axx buscam as cotações e as organizam para estudo futuro
        fonte: B3
Os arquivos com prefixo bxx buscam os dados de balanço e organizam para estudo futuro
        Fonte: CVM
